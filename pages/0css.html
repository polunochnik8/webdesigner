• О CSS! Или язык оформления внешнего вида HTML.

CSS (/сеːɛsɛs/ англ. Cascading Style Sheets — каскадные таблицы стилей или неофициальная версия"Каскадный стиль дерьмо") язык изначально в основном использовавшийся в структуре документа html, который позднее начали выносить в отдельный документ в связи с его усложнением и увелечением размеров. Данный язык отвечает за расширенные настройки внешнего отображения html документа. Термин «каскадные таблицы стилей» был предложен Хоконом Ли в 1994 году. Совместно с Бертом Босом он стал развивать CSS. Сам же css зародился в 1991 году, становление которого произошло лишь в 1996. Язык появился из-за разного подхода браузеров к интропретации страниц HTML, что стало одной из причин "Браузерных войн", а язык css должен был решить вопрос одинаковой интропретации во всех браузерах, с помощью которого устанавливались конкретные установки.


CSS оброс (свойствами) атрибутами и широко используется для оформления интернет страниц, термин же вёрстка перекочевал из издательских сфер — типографий и полиграфий.


CSS 3 — последняя версия языка, мнения по поводу которого разделились и который до сих пор не внедрён. В целом история похожа на историю развития HTML, и язык преследуют похожие проблемы, в результате чего ввелись новые стандарты измерения типа "rem" "em" и т.д., которые якобы лучше решают насущные проблемы экранов особо высокого разрешения от Аpple. Некоторые элементы языка считаются эксперементальными или спорными и не поддерживаются всеми браузерами, что в некоторых случаях усложняет оформление приверженцами эксперементальных функций, что вызывает утяжеление веса документа css, в результате чего придумали "сокращённый вариант" min.css, в котором не используются интервалы и пробелы между селекторами с свойствами и значениями, которые повсеместно используются в библиотеках фреймворков css.


Примечательно, что в нашей стране ни разу не издавался справочник по CSS, который бы конкретно разделялся на спецификации CSS 1, CSS 2, CSS 3. Вероятно был бы интересен похожий справочник по HTML ( так мы бы увидели историю развития и изменения данных языков).
